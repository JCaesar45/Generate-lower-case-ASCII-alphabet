# Aether LAScii

**The luxury lowercase ASCII range generator.**

An ultra-premium single-page experience that turns a simple utility into performance art. Built to impress CTOs, principal engineers, red-team hackers, and design-obsessed clients in equal measure.

## Features
- Live canvas particle field with reactive lowercase glyphs
- Instant range generation with burst animation
- Glassmorphic UI with custom properties and scroll-driven polish
- Three-language showcase (vanilla JS, Python 3, Java 17+)
- Copy-to-clipboard CTAs that feel premium
- Fully responsive, zero dependencies, sub-2kb gzipped

## Quick Start
1. Open `index.html`
2. Type any lowercase letters (e.g. a → d)
3. Watch the sequence manifest

## Tech Stack
- HTML5 semantic + Canvas
- CSS3 (custom properties, container queries, backdrop-filter, advanced keyframes)
- Vanilla ES6+ JavaScript (modules pattern, requestAnimationFrame, DOM orchestration)
